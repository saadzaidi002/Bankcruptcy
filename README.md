# 🏦 Corporate Bankruptcy Prediction System

This project is an advanced, end-to-end Machine Learning web application designed to predict the likelihood of corporate bankruptcy. Built with a sleek, premium UI using **Streamlit**, it leverages powerful predictive models including **Artificial Neural Networks (ANN)** and **Random Forests** to analyze financial indicators and deliver accurate risk assessments.

## 🚀 Features
- **Interactive UI:** A highly polished, modern glassmorphic interface with smooth navigation.
- **Multiple ML Models:** Choose between Artificial Neural Networks, Random Forest, or Support Vector Machines for predictions.
- **Real-time Assessment:** Input financial metrics directly into the app for immediate bankruptcy risk evaluation.
- **Exploratory Data Analysis:** Built-in data visualization tools to understand financial trends.
